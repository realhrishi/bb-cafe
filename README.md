# ☕ BB Cafe

A modern, responsive café website built with **React**, **Vite**, and **Tailwind CSS**, featuring a premium user interface, smooth animations, an interactive menu, and an online reservation system.

## 🌐 Live Demo

**Website:** https://bb-cafe-theta.vercel.app/

---

## ✨ Features

* 🎨 Modern and responsive UI
* 📱 Fully optimized for desktop, tablet, and mobile devices
* 🍽️ Interactive food & beverage menu
* 📅 Table reservation section
* ⭐ Customer testimonials
* 📍 Contact & location section
* ⚡ Fast loading with Vite
* 🎭 Smooth animations and transitions
* 🌙 Elegant café-inspired design

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/BB-Cafe.git
```

### Navigate to the project

```bash
cd BB-Cafe
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```
